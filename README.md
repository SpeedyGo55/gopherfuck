# Gopherfuck

Gopherfuck is a Brainfuck interpreter written in Go (Golang). Its goal is to provide an efficient and easy-to-use solution for interpreting and executing Brainfuck code while showcasing the simplicity and power of Go.

## Features

- Full Brainfuck language support
- Written in Go, focusing on simplicity and performance
- Lightweight and minimal

## Installation

1. Ensure that you have [Go](https://go.dev/) installed on your machine.
2. Clone the repository:

   ```bash
   git clone https://github.com/SpeedyGo55/gopherfuck.git
   ```

3. Navigate to the project directory:

   ```bash
   cd gopherfuck
   ```

4. Build the interpreter:

   ```bash
   go build
   ```

## Usage

1. After building the binary, run Gopherfuck with a Brainfuck source file as input:

   ```bash
   ./gopherfuck [path-to-brainfuck-source-file]
   ```

2. Example:

   ```bash
   ./gopherfuck bf_tests/hello_world.bf
   ```

   This command will execute the `hello_world.bf` Brainfuck program in the examples directory.
